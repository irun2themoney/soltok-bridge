# Extension Icons

Create the following PNG icons for the extension:

- icon16.png (16x16)
- icon48.png (48x48)  
- icon128.png (128x128)

The icons should feature the SolTok Bridge logo - a bridge symbol with Solana/crypto styling in emerald green (#10b981).

For development, you can use placeholder images or generate them from the SVG below:
